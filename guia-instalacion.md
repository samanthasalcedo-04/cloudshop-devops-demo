# Guía de instalación

## Requisitos previos
Para trabajar con el proyecto CloudShop, asegúrate de cumplir con lo siguiente:
- Tener instalado **Git** en tu sistema operativo.
- Contar con una cuenta activa en **GitHub**.
- Tener configuradas las credenciales de usuario (nombre y correo) en tu entorno local.
- Acceso a la terminal o consola de comandos.

## Pasos de instalación
1. **Clonar el repositorio:**
   Ejecuta el comando: `git clone https://github.com/samanthasalcedo-04/cloudshop-devops-demo.git
2. **Acceder al directorio:**
   Entra a la carpeta usando: `cd cloudshop-devops-demo`
3. **Configuración inicial:**
   Crea una nueva rama para tus cambios: `git checkout -b feature/guia-instalacion`

## Validación
Para confirmar que la instalación y el entorno están listos:
- Verifica que te encuentras en la rama correcta ejecutando: `git branch`
- Asegúrate de que el archivo `README.md` es visible en el directorio listando los archivos
